# AI-Assisted Programming Assignment Project

## Selected Problem

The selected moderate-complexity problem is a Library Management System. The system allows a librarian to register books and members, issue books, return books, search records, and view current borrowing status.

## Project Structure

- `source_code/version_a_traditional/library_system.py` - traditional baseline implementation.
- `source_code/version_b_ai_assisted/library_system_ai.py` - AI-assisted/refined implementation.
- `diagrams/` - UML diagrams used in the report.
- `screenshots/` - sample run screenshots used in the report.
- `AI_Assisted_Programming_Report.docx` - editable report.
- `AI_Assisted_Programming_Report.pdf` - final report PDF.

## How to Run

Version A:

```bash
python source_code/version_a_traditional/library_system.py
```

Version B:

```bash
python source_code/version_b_ai_assisted/library_system_ai.py
```

Version B creates `library_data.json` automatically in its own folder to persist data between runs.
